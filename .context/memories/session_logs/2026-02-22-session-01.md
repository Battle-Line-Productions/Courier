## 2. Key Decisions & Insights (The Minutes)
*   **Decision**: Custom Courier color theme — dark charcoal sidebar (oklch 0.165) with warm amber primary (oklch 0.42, hue 55) instead of generic shadcn defaults. Industrial-utilitarian aesthetic.
*   **Decision**: Used `api.replaceSteps()` for create flow instead of sequential POST calls — cleaner atomic operation.
*   **Decision**: Named API error class `ApiClientError` to avoid conflict with the `ApiError` type interface.
*   **Insight**: `npx shadcn@latest add` with multiple components silently fails if `toast` (deprecated) is included. Install individually or use `sonner` instead.
*   **Insight**: `shadcn init` overwrites globals.css — need to re-apply custom theme after init.
*   **Decision**: Filesystem error codes go in 3000-3999 range (next band after Jobs 2000-2999).
*   **Decision**: FilesystemService is a simple I/O wrapper with no DB dependency — registered as scoped to match other services, but stateless.
*   **Decision**: File browser dialog uses a single `browseTarget` state ("source"|"destination"|null) rather than two separate boolean states — cleaner toggle pattern.
