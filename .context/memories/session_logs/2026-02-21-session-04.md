## 2. Key Decisions & Insights (The Minutes)
*   **Decision**: Use `Seq__ServerUrl` custom env var instead of polluting Serilog's `WriteTo` config namespace
*   **Decision**: Use `WithHttpEndpoint(env: "PORT")` without hardcoded port for JavaScript apps in Aspire — let DCP assign the port dynamically
*   **Decision**: Add permissive CORS default policy for development (AllowAnyOrigin/Header/Method)
*   **Insight**: Aspire AppHost requires `Properties/launchSettings.json` for the dashboard to auto-launch
*   **Insight**: Aspire needs `UserSecretsId` in csproj to persist auto-generated Postgres passwords across runs — without it, each launch generates a new password that mismatches the persistent Docker volume
*   **Insight**: `WithHttpEndpoint(port: X, env: "PORT")` creates a DCP proxy on port X — if the app also hardcodes that port via `-p X`, there's a silent conflict causing blank pages
*   **Insight**: Next.js `next dev` respects the `PORT` env var when no `-p` flag is specified
